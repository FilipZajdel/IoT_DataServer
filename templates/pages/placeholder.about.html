About AGH ENV Sense

Goals

This project was developed to gain knowledge and skills of NB-IoT usage and integration of NB-IoT based sensors network with web platform.


Technology

To send sensor readings NB-IoT is used. Each reading is sent via HTTP, which saved a lot of time to accomplish the solution. However using HTTP over NB-IoT is not considered a good practice, since it needs to send a lot of additional data.
